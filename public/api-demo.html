API Endpoints

Token Analysis

POST

Endpoint: /api/token/analyze

Analyze any token contract address and get AI-generated insights

Request Example:
{ "contractAddress": "0x1234567890abcdef..." }
Response Example:
{ "success": true, "analysisId": "analysis_123", "narrative": "This token shows strong fundamentals...", "riskAssessment": "Medium risk" }

Chain Analysis

POST

Endpoint: /api/chain/analyze

Get comprehensive analysis of OKX Chain trends and market data

Request Example:
{ "analysisType": "trend" }
Response Example:
{ "success": true, "chainData": { "totalTransactions": 1500000, "activeWallets": 45000 }, "trends": { "marketSentiment": "bullish" } }

Response Example:
